# Madhav_Store_PowerBI_Dashboard
Project Overview

This project is a professional Power BI dashboard built using retail store sales data. The dashboard provides valuable insights into sales performance, customer behavior, profitability, and business trends through interactive visualizations.

🎯 Objectives
Analyze overall sales performance
Track profit and revenue trends
Identify top-selling products and categories
Understand customer purchasing behavior
Support data-driven business decisions
🛠️ Tools & Technologies
Power BI
Power Query
DAX (Data Analysis Expressions)
Microsoft Excel / CSV Dataset
📈 Dashboard Features
Interactive Sales Dashboard
Revenue & Profit Analysis
Category-wise Sales Performance
Product-wise Performance Tracking
Customer Analysis
Monthly & Quarterly Trends
Dynamic Filters and Slicers
KPI Cards for Key Metrics
📊 Key Insights
Identified top-performing product categories
Tracked monthly sales and profit growth
Analyzed customer purchase patterns
Highlighted high-revenue products
Provided actionable business insights
🚀 How to Use
Download the .pbix file.
Open it in Power BI Desktop.
Refresh the dataset if required.
Explore the dashboard using interactive filters and visuals.
